
## Svg Line element

As the name applies, it is used to draw lines.

```html
<line x1="200" x2="150" y1="50" y2="150" stroke="pink" />
```

![line](img/line.png)

| attribute | meaning                    |
| --------- | -------------------------- |
| x1        | The x position of point 1. |
| y1        | The y position of point 1. |
| x2        | The x position of point 2. |
| y2        | The y position of point 2. |
